# Assignment Project.



## How to Run
Install all dependencies
### `npm i`
Start Client Dev Server
### `npm start`
Goto backend directory
### `cd .\backend\`
Create .env file, set USERNAME and PASSWORD and
Start Server
### `node .`
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
